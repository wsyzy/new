发表于 2020-06-23
字数统计:

环境配置

git和node.js

git : 开源分布式版本控制系统
Node.js : 运行在服务端的 JavaScript

git

mac上可以直接用Homebrew装git, 当然 得先装Homebrew
ssh私/公匙 github上配置公匙

node.js

用nvm安装

安装hexo

使用Node.js 中的npm包管理工具,将hexo这个模块装到电脑中

最后

分别用v来检查是否安装成功
权限问题试试 sudo 或者忽略安全方面的警告强制安装

使用Hexo(查看官方文档)

创建你的blog文件 , 也称 建站

切入你准备用来搭建blog的目录
先执行 hexo init
再执行 npm install

预览本地博客

仍在你的blog目录下
执行 hexo start
浏览器打开 http://localhost:4000/ 预览, ctr+c 停止运行

部署到github上

github上的操作

github上新建一个 repository, 注意命名的固定模式为: userName.github.io
配置公匙在前面说了, 检查一下

hexo部署

在blog目录下运行 hexo g -d
访问 https://用户名.github.io 就可以看到你的远程博客了
